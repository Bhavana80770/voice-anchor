## 🎙️ Murf AI Integration

Voice Anchor uses **Murf AI as the core therapeutic feature**:

| Setting | Value | Reason |
|---------|-------|--------|
| Voice | `en-US-natalie` | Warm, soft, human-like female voice |
| Style | `Calm` | Specifically designed for ASD children |
| Rate | `-10` | Slightly slower for better comprehension |
| Model | `GEN2` | Murf Falcon — under 130ms response time |
| Format | `MP3` | Direct browser playback |

## 🎬 How It Works

```
1. Caregiver creates account and logs in securely
   ↓
2. Sets up child profile — name, age, photo, favourite thing
   ↓
3. Child selects current emotion — Happy, Sad, Angry, etc.
   ↓
4. Caregiver taps routine button — Stop Playtime, Brush Teeth, etc.
   ↓
5. Claude AI generates warm personalized calming script
   ↓
6. Murf AI speaks it in soothing calm voice under 130ms
   ↓
7. Breathing animation plays — child hears their name and calms down
   ↓
8. Session saved — progress tracked — star earned
```
